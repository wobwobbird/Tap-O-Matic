## âœ¨ Features

### ğŸ® Games & Tools

- **Battle For 100** - Multi-player competitive game where players select numbers 1-9. Random numbers are generated until one player reaches 50 points. Features real-time score tracking with visual sliders.

- **The Selector** - Choose a number range and player selections. A random number is generated from the range, and the player whose number matches wins. Includes visual number range display for smaller ranges.

- **Closest To 21** - A Blackjack-inspired game where two players take turns rolling dice (1-9). Players can choose to roll or pass each turn. The player closest to 21 without going over wins!

- **Coin Flip** - Simple heads or tails coin flip with smooth animations. Select your choice and watch the coin spin to reveal the result.

- **The Three Kasias** - A hidden easter egg screen unlocked by tapping the "The Three Kasias" card on the home screen 5 times. Features an infinite scrolling list with alternating color schemes.
